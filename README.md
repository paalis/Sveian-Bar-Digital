# Sveian pilot

Dette er en GitHub Pages-klar pilot for Sveian bestilling + barskjerm.

## Sider

- Kundeside: `#/bestill`
- Barskjerm: `#/bar`

## Publisering på GitHub Pages

1. Last opp `index.html` til repoet.
2. Gå til Settings > Pages.
3. Velg `Deploy from a branch`.
4. Velg `main` og `/(root)`.
5. Åpne GitHub Pages-lenken.
